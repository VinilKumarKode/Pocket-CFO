# Pocket CFO

Version: 0.1 Genesis

Author:
Vinil K

Project Type:
Offline Android Application

Mission:
Pocket CFO is an intelligent offline personal financial management application that helps users manage income, expenses, loans, investments, insurance, bills and credit cards while providing AI-driven financial insights.
